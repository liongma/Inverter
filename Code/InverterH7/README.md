**50 kW Inverter Firmware**
- Includes Advanced Control architecture based on DBCC with ESO, one step delay compensation, Hybrid RLS flux parameter estimation
- Advanced Fault detection based on kalman filters, and taylor state approximations
- MTPA, MTPV algorithms based on flux optimization 
- Dynamic Field weakening integrated into current/speed controllers
